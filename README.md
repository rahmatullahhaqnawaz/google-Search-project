# Google Search Front-End

A front-end web project that recreates Google Search, Google Image Search, and Google Advanced Search using HTML and CSS. The project submits user queries via GET parameters to Google’s search endpoints, focusing on form handling, query strings, and clean UI design inspired by real-world interfaces.

## Features
- Google Search
- Google Image Search
- Google Advanced Search
- “I’m Feeling Lucky” functionality
- Google-like layout and styling

## Technologies Used
- HTML5
- CSS3

## What I Learned
- How HTML forms communicate with servers using GET requests
- How query parameters are structured and parsed
- Building multi-page front-end applications
- Recreating real-world UI behavior and layouts
- Understanding how front-end interfaces interact with existing backends

## Project Context
This project was completed as part of **CS50’s Web Programming with Python and JavaScript (CS50W)**.
